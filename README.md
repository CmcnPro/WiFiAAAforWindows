# WiFiAAA for Windows
===

成都东软学院第三方网络客户端 For Windows

## What's AAA

AAA 是计算机网络安全术语 认证(Authentication)、授权(Authorization)、计帐(Accounting) 的缩写。

## What's WiFiAAA

WiFiAAA 是基于OpenAAA协议开发的第三方网络客户端

根据相关规定，网络上开源的版本不会加入开启承载网络的功能！
